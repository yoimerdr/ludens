## Why Ludens?

Ludens lets you take your existing RPG Maker MV/MZ game and wrap it in a native mobile application powered by Kotlin and Compose Multiplatform. No need to rewrite your game — just export, configure, and build.

<CardGrid>
  <Card title="Optimized WebView" icon="laptop">
    Seamlessly integrates your HTML5 RPG Maker game into a native mobile shell
    with full WebView support.
  </Card>
  <Card title="On-Screen Controls" icon="setting">
    Built-in virtual joystick and configurable buttons (A, B, X, Y) with
    adjustable opacity and positions.
  </Card>
  <Card title="Settings Screen" icon="list-format">
    Complete settings UI: theme, language, audio mute, FPS display, WebGL
    toggle, control layout, and quick actions.
  </Card>
  <Card title="Easy Configuration" icon="document">
    Customize your app's ID, version, name, and icon directly from
    `gradle.properties` — no Kotlin code changes needed.
  </Card>
</CardGrid>
