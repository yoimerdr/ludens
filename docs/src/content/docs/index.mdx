## Why Ludens?

Ludens lets you transform your existing game into a professional mobile application for Android and iOS. Powered by Kotlin and **Compose Multiplatform**, it eliminates the need to rewrite your code — just export, configure, and build for your target platform.

This keeps your original game logic while adding native mobile delivery, touch controls, and app-level settings.

  <Card title="Optimized WebView" icon="laptop">
