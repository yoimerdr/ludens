## ¿Por qué Ludens?

Ludens te permite transformar tu juego existente en una aplicación móvil profesional para Android e iOS. Impulsado por Kotlin y **Compose Multiplatform**, elimina la necesidad de reescribir tu código — solo exporta, configura y compila para tu plataforma de destino.

Esto mantiene la lógica original de tu juego mientras añade distribución móvil nativa, controles táctiles y ajustes a nivel de aplicación.

<CardGrid>
  <Card title="WebView Optimizado" icon="laptop">
    Integra sin problemas tu juego HTML5 de RPG Maker en un shell móvil nativo con soporte completo de WebView.
  </Card>
  <Card title="Controles en Pantalla" icon="setting">
    Joystick virtual integrado y botones configurables (A, B, X, Y) con opacidad y posiciones ajustables.
  </Card>
  <Card title="Pantalla de Ajustes" icon="list-format">
    Interfaz completa de ajustes: tema, idioma, silenciar audio, mostrar FPS, alternar WebGL, diseño de controles y acciones rápidas.
  </Card>
  <Card title="Configuración Fácil" icon="document">
    Personaliza el ID, versión, nombre e icono de tu app directamente desde `ludens.properties` — sin cambios de código Kotlin.
  </Card>
